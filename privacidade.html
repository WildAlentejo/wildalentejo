Política de Privacidade

Última Atualização: Dezembro de 2025

1. Dados Recolhidos pelo Website

O website Wild Alentejo é um site de documentação **estático** alojado no GitHub Pages.

  • Não recolhemos dados pessoais. Não utilizamos formulários de contacto nem armazenamos bases de dados de visitantes.
  • Não utilizamos *cookies* de rastreio de terceiros para fins publicitários ou de marketing.
  • A navegação é anónima.

2. Ligações e Serviços Externos

O nosso website pode conter ligações para plataformas e serviços externos (ex: Blogger, se usar análise de tráfego). Qualquer interação nessas plataformas é regida pelas políticas de privacidade desses serviços.
